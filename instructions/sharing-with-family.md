# Sharing Your Plan with Family

There are three ways to share your emergency plan with family members. Pick the one that works for your situation.

---

## Option 1 — ZIP file (easiest, works offline)

Best for: sending a copy to someone right now, no accounts needed.

**To send your plan:**
1. Open the app → **Settings** → **Share or back up your plan**
2. Click **Export as ZIP**
3. Send the downloaded `.zip` file to your family member (email, iMessage, AirDrop, etc.)

**For the recipient to open it:**
1. Open [https://jpwill00.github.io/family-prepared](https://jpwill00.github.io/family-prepared) in any browser
2. Click **Restore from a saved file**
3. Select the `.zip` file you received

That's it — the full plan loads on their device.

---

## Option 2 — Online backup with a shared repository (live sync)

Best for: multiple family members who each want to read and update the same plan.

**Setup (one person does this once):**
1. Open the app → **Settings** → **Online backup (optional)** → **Set up online backup**
2. Complete the one-time authorization — the app creates a private backup repository for you automatically
3. Note the repository name (shown in Settings after connecting, e.g. `yourname/family-prepared-yourname`)
4. Go to that repository on GitHub → **Settings** → **Collaborators** → invite each family member

**For each family member to join:**
1. Accept the GitHub collaborator invite
2. Open the app → click **Sign in to restore from cloud backup**
3. Complete the authorization flow
4. When asked, click **I already have a backup repository** and enter the shared repository name (e.g. `yourname/family-prepared-yourname`)

Each person can now save changes with **Save to cloud now** and load the latest version with **Load latest from cloud**.

> **Note:** The last person to save wins. For most families this is fine — you're unlikely to be editing simultaneously.

---

## Option 3 — Screenshot or PDF (print-friendly)

Best for: printing a physical copy for a go-bag, fridge, or safe.

1. Open the app → **Settings** → **Share or back up your plan**
2. Click **Export PDF binder**
3. Print the downloaded PDF or save it to a USB drive

The PDF includes all filled-in sections in a readable format designed for printing.
